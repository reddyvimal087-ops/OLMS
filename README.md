
# Offer Letter

This repository contains a design-driven offer letter application with a React frontend and a minimal Express backend.

## Structure

- `frontend/` — React + Vite application using the provided Figma-derived design.
- `backend/` — Node.js + Express API for persisting profile and document state in PostgreSQL.
- `database/migrations/` — SQL migrations for the required database schema.
- `docs/` — setup and usage documentation.
